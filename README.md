# ⭐ TAFEL STAR - Arena Battle Game

Een interactief educatief spel waarbij je tafels oefent terwijl je in een arena vecht!

## 🎮 Spelen

Open `index.html` in je browser en klik op **SPELEN** om te beginnen!

## 🎯 Hoe te spelen

1. **Beweeg** je karakter met WASD of pijltjestoetsen
2. **Beantwoord** de rekensom die verschijnt om schade uit te delen
3. **Vermijd** vijanden en blijf zolang mogelijk in leven
4. **Verzamel** coins en trophies om sterker te worden

## 🧮 Rekensysteem

- **Goed antwoord** = 1.5x schade aan vijanden ✅
- **Fout antwoord** = 0.5x schade aan vijanden ❌
- Hoe sneller je antwoordt, hoe sterker je aanval!

## 🦸 Brawler System

Je karakter wordt sterker naarmate je speelt:
- Verzamel trophies om op te klimmen in ranks
- Unlock nieuwe content bij hogere levels
- Upgrade je HP en damage met coins

## 💾 Opslaan

Je voortgang wordt automatisch opgeslagen in je browser (localStorage).

## 📊 Progressie

- **Coins**: Verdien door vijanden te verslaan
- **Trophies**: Win door matches te winnen
- **Level**: Stijg door trophies te verzamelen
